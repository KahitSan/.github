## Getting Started

### Prerequisites
- Node.js 20+ (LTS recommended)
- Bun 1.2.22+ (for PillarBridgeAPI)
- Git with SSH access

### Contributing

We welcome contributions to our projects. Each repository has its own setup instructions and contribution guidelines.

**See our [Contributing Guide](../CONTRIBUTING.md) for:**
- General workflow and commit conventions
- How to set up your development environment
- Repository-specific guidelines
- Testing and code quality standards

**Note:** You'll only be able to contribute to repositories you have access to. If you don't see a repository listed in the organization, you may not have the necessary permissions.

**Testing:** Bun:test, Vitest, Playwright
